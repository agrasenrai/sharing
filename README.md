# Private High-Quality Media Sharing Platform

This repository contains a minimal scaffold for a private media sharing platform.

Structure:
- `backend/` — Express server handling auth and Cloudinary uploads
- `frontend/` — React (Vite) app with upload and gallery pages

Next steps:
1. Create a Cloudinary account and set credentials in `.env`.
2. `cd backend && npm install` then `npm run dev` to start the backend.
3. `cd frontend && npm install` then `npm run dev` to start the frontend.
